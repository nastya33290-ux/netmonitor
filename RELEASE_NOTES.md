# NetMonitor v1.0.0

Первый релиз кроссплатформенной программы для мониторинга сетевых соединений и HTTP(S) запросов.

## Скачать
- Готовый исполняемый JAR: [`dist/netmonitor.jar`](dist/netmonitor.jar) (нужна Java 17+)
- Запуск: `java -jar netmonitor.jar`

## Возможности
- **Мониторинг соединений** всего ПК с привязкой к процессам (PID + имя), фильтр по выбранному приложению, поиск, авто-обновление. Windows / Linux / macOS.
- **HTTP(S) прокси** с перехватом запросов: метод, URL, заголовки, тело, статус и размер ответа.
- **Блокировка** запросов по хосту (glob `*.ads.com`), URL (подстрока/regex), методу и направлению.
- **Отправка собственных запросов** (мини-Postman), опционально через свой прокси.
- **Настройки** (порт, интервалы, уровень логов, автозапуск) и **панель логов/отладки**.
- **Консольный режим**: `--cli connections | send | proxy`.

## Известные ограничения
- HTTPS перехватывается в режиме туннеля: блокировка и лог по хосту, без расшифровки тела (нужен MITM-сертификат — не входит в этот релиз).
- Вкладка соединений использует системные утилиты ОС (`netstat`/`ss`/`lsof`) — они должны быть доступны в системе.
